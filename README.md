# 🚀 Instahyre Auto Apply Chrome Extension

A Chrome extension that automates job applications on Instahyre by detecting and clicking "Apply" buttons intelligently.

---

## ✨ Features

- ✅ Auto apply to jobs with one click
- 🔢 Configurable number of jobs to apply
- 🔄 Auto-run on page load
- ⚡ Works with dynamic pages (SPA handling)
- 🧠 Smart detection of apply buttons
- ⏱️ Delayed execution to avoid spam

---

## 🛠️ Tech Stack

- JavaScript (Vanilla JS)
- Chrome Extension APIs (Manifest v3)
- DOM Manipulation

## ⚙️ Installation (Local Setup)

1. Clone or download the repository
2. Open Chrome and go to:
   chrome://extensions
3. Enable Developer Mode
4. Click Load Unpacked
5. Select the project folder

---

## ▶️ How to Use

1. Open Instahyre
2. Click on the extension icon
3. Set:
    - Number of jobs to apply
    - Enable auto-run (optional)
4. Click Apply Now

---

## 🔁 How It Works

- Injects script into Instahyre pages
- Detects "Apply" / "Interested" buttons
- Clicks them with delay
- Handles dynamic page navigation (SPA)

---

## ⚠️ Disclaimer

This extension is intended to assist users with repetitive actions.  
Use responsibly and avoid excessive automation that may violate platform policies.
